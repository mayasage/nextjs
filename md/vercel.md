# Vercel

## Deploying Hokusai

For Hokusai, we'll need 2 Github Repositories:

- JsonServ: to serve as dummy API server
- Hokusai: app repository

Why ?

Because, vercel will only start our NextJS application.
It won't use concurrently to start both JsonServ & Hokusai.
Therefore, we have to deploy the JsonServ separately, and use that API link
inside Hokusai, instead of localhost.

### Deploying JsonServ

[Docs](https://github.com/typicode/json-server/tree/v0#deployment)

- Create a new Github Repository (`jsonServer`)
- `npm i --save-exact json-server@0.17.4`

const jsonServer = require("json-server");
const server = jsonServer.create();
const router = jsonServer.router("db.json");
const middlewares = jsonServer.defaults();

server.use(middlewares);
server.use(
